# 🏰 Voy Disney - Agencia Boutique

Landing page y aplicación web para **Voy Disney**, una agencia de viajes boutique (por Agus & Marti) especializada en diseñar experiencias mágicas en Disney World, Disneyland, Universal Studios y Disney Cruise Line.

## ✨ Características Principales

* **Diseño Premium y Mágico:** Interfaz moderna, limpia y temática utilizando una paleta de colores personalizada (Disney Pink, Ink Soft, etc.).
* **Navegación Fluida (Smooth Scroll):** Menú de navegación anclado que resalta la sección activa y se desplaza suavemente por la página.
* **Componentes Modulares:** Arquitectura basada en componentes reutilizables (Hero, Services, Testimonials, InstagramFeed, Footer).
* **Modales Interactivos:** Tarjetas de servicios que abren modales detallados con galerías de imágenes y listas de inclusiones.
* **Integración con Google Sheets:** Formulario de contacto y cotización conectado directamente a un Excel (Google Sheets) mediante Google Apps Script.
* **Llamados a la Acción (CTA):** Botones flotantes de WhatsApp y CTAs estratégicos para maximizar la conversión de clientes.
* **Animaciones Fluidas:** Transiciones y efectos de *hover* impulsados por Framer Motion.

## 🛠️ Tecnologías Utilizadas

* **Framework:** [React 18](https://reactjs.org/)
* **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
* **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
* **Iconos:** [Lucide React](https://lucide.dev/)

## 🚀 Instalación y Uso

### Contacto / Backend (Google Apps Script)
El formulario de contacto envía datos en formato JSON mediante un método POST a una Web App de Google Apps Script. El script valida los datos y los añade como una nueva fila en "Hoja 1".

Diseñado y desarrollado con ❤️ para familias viajeras.
